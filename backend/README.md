## Installation
* Create postgress database.
* Create a user (see `migrations/manual/01-init.sql`).
* `npm run publish-migration`.
* Add user (admin / Password2) (see `migrations/manual/02-seed.sql`).
* install dependencies `npm i`

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```
